# Bangalore House Price Insights

An end-to-end data analytics project that analyzes Bangalore housing prices to uncover
location-wise trends, price per square foot patterns, and key factors influencing real
estate prices.

This project combines **Python-based data analysis** with an **interactive Power BI dashboard**
to deliver clear, business-focused insights.
